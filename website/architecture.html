§ 04 — STATISTICAL SIGNAL LAYER

Signal Mathematics

Kalman Filter (1D Scalar) x̂ₖ = x̂ₖ₋₁ + K·(zₖ − x̂ₖ₋₁)
K = P⁻·(P⁻ + R)⁻¹
Optional db4 wavelet pre-denoising · outputs: trend, slope, noise_ratio, signal ∈ {−1,0,+1}
Hurst Exponent (R/S Analysis) H = log(R/S) / log(n)
H > 0.6 → trending · H < 0.4 → mean-reverting
100-bar rolling window · OLS log(RS) ~ log(lag) · regime gate for swing entries
Half-Life (Ornstein-Uhlenbeck) Δpₜ = α + β·pₜ₋₁ + εₜ
HL = −ln(2) / ln(1 + β)
Gates BB mean reversion entries · valid HL ∈ [1, 30] bars · β ∈ (−1, 0) required
Half-Kelly Position Sizing f* = (p·b − q) / b
size = (f*/2) · equity · risk%
90-day lookback from signal_outcomes · min 20 closed trades · 10% hard cap · 2% default
